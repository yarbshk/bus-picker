# BusPicker
A simple JavaFX application that filter a list of objects (buses) from a JSON file.

### Getting started
To run the application from the sources:\
`$ ./gradlew jfxRun`

To pack a self-contained application into the `build/jfx/naive` directory:\
`$ ./gradlew jfxNative`
